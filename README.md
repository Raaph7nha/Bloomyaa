#  Descripción

**BLOOMY** es una plataforma inteligente enfocada en la gestión y reconocimiento de plantas, diseñada para ayudar a los usuarios a identificar especies, organizar su colección y aprender sobre el cuidado adecuado de cada planta.


---

#  Características Principales

 Escaneo e identificación de plantas  
 Catálogo interactivo de especies  
 Gestión personalizada de plantas  
 Búsqueda dinámica de plantas  
 Diseño moderno y responsivo  
 Base de datos
 Despliegue online con Render
